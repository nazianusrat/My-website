IOT Based Online Monitoring System For Transformer

Power Transformer is the main component of electric power distribution. Therefore monitoring transformer health and conditions has immense importance. The main objective of the project is to monitoring the condition of power transformer located from remote location. For this purpose the four main criteria of transformer which are Voltage, current, temperature & oil level of the transformer has been taken. To achieve the goal, this project was implemented by using Arduino, GSM module, & Arduino compatible Ethernet shield. If any one of the parameter i.e. voltage, current, temperature & oil level crossed their threshold, will be notified by SMS using Arduino and GSM module. Similarly same data will be uploaded to a server for online monitoring using Arduino & Ethernet shield.

Intelligent Greenhouse

Green house is the most convenient way to achieve sustainable food production. This is the design of a sensor network that provides real-time monitoring of temperature, humidity, level of carbon dioxide, light intensity and soil moisture of a greenhouse. An automated control system for managing these micro-climate parameters is developed to optimize the parameters. The sensor node developed handles the data from the sensors and triggers actuators based on the threshold algorithm programmed into the microcontroller. The gateway receives the sensor data and control information through Arduino Mega and transmits the data to the actuators for remote monitoring. The monitor software provides network view with nodes and their information. Information management system is also designed to monitor the data at any required time.

Bluetooth Controlled Home Automation System

Automation system in home is an exciting field of work, where new technology and technique is required on a regular basis. There are loads of methods to implement a home automation system. But every method has common term and that is wireless control. Wireless control can be achieved in several ways like Bluetooth, Wi-Fi etc. But everything above, a computer is necessary to make all of this happen. In this project a miniature computer has been used i.e. microcontroller (arduino). This is programmable according to user requirements. And can be modified even after implementation. This project shows an implementation microcontroller-Bluetooth based system which can be controlled by an android phone. The communication between the cell phone and the Arduino BT board which is wireless.
